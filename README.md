# Truck HOS Planner

Full-stack application to plan truck routes and generate ELD logs based on Hours of Service (HOS) regulations.

## Tech Stack
- Backend: Django + Django REST Framework
- Frontend: React + TypeScript

## Current Features
- Basic trip planning form
- API connection between frontend and backend
- Mock trip results display

## Next Steps
- Integrate map API for real distance calculation
- Implement HOS rules engine
- Generate ELD log sheets
